#  Random Quote Machine

A modern Random Quote Machine built with React for the FreeCodeCamp Front End Development Libraries Certification. The application displays inspirational quotes, allows users to generate a new random quote, and share the current quote directly on Twitter.

## Features

-  Generate random inspirational quotes
-  Display the quote author
-  Share quotes directly on Twitter
-  Built with React
-  Responsive design

##  Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/fcc-random-quote-machine.git
```

Go to the project directory:

```bash
cd fcc-random-quote-machine
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
